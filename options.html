Omnibar

v1.1.0

Keyboard Shortcut

Press ⌘ + K to open Omnibar on any page.

To change this shortcut, go to your browser's extension settings and configure keyboard shortcuts.

Usage

Tab Switch between Tabs, History, and Web search
↑ ↓ Navigate through results
Enter Select result
Esc Close Omnibar

Made for Orion and Chrome
